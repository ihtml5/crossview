crossview basic tool library

兼容web，小程序，nodejs等

一、Install using npm

npm install crossview-utils --save

二、特点

  • 多端支持包括web，小程序，nodejs
  • 编写一次，多处运行
  • 简单可扩展
  • 可帮助你快速搭建可复用的多端函数库
  • 集成web和小程序等项目开发中常见功能，提升开发效率
